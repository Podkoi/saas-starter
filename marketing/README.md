# Marketing Assets

This folder contains starter marketing materials for selling the SaaS Starter template.

- `pitch-deck.md` — A 7-slide pitch deck in Markdown (convert to PPTX or PDF as needed).
- `one-pager.md` — Short product one-pager.
- `screenshots/` — Example SVG screenshots (hero and dashboard) to include in listings.

Next steps

- Convert `pitch-deck.md` to PowerPoint or PDF using your preferred tool.
- Replace SVGs with real screenshots from the running app (use `npm run dev` and capture images).
